# Learning Loop Framework

## Goal

Provide a low-friction daily English system that supports overseas work-life communication through QQ.

## Skill loop design

1. Goal:
- Define a 2-4 week target (`/learn goal ...`)

2. Content focus:
- Define a practical domain (`/learn content ...`)
- Example: standup, one-on-one, apartment renting, doctor visit

3. Plan:
- Pull daily checklist (`/learn plan`)
- Keep plan short enough to finish in 15 minutes

4. Execute:
- Practice one message at a time (`/learn do ...`)
- Return correction, rewrite, and next action

5. Review:
- Log completion (`/learn checkin ...`)
- Capture reflection (`/learn review win|blocker|nextAction`)

6. Improve:
- Inspect progress (`/learn status`)
- Generate targeted optimization (`/learn improve`, `/learn week`)

## Online case patterns used

1. Duolingo pattern:
- Strong daily habit loop (daily goal, streak, short tasks)
- Action used in this skill: concise daily checklist + completion signal

2. Duolingo Max pattern:
- Roleplay + explanation workflow
- Action used in this skill: simulate scenario turn, then provide correction and coaching

3. CEFR-aligned communication target:
- Real-world communication ability focus rather than isolated grammar drills
- Action used in this skill: prioritize practical scenario outcomes

## Evidence-informed principles

1. Specific goals improve performance commitment.
2. Spaced/distributed practice improves retention.
3. Deliberate practice plus immediate feedback speeds skill acquisition.
4. Self-regulated cycle (plan, perform, reflect) improves persistence.
